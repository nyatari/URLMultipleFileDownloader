### Multiple Files Downloader from base URL
The code in main.py represent a very simple way to download files one after another with python using the requests library
<br>
First replace the variable `url` with a designed base url
<br>
Then replace the `filetype` variable with the designed type of the files being downloaded
<br>
Lastly replace the the number on the `downloadRange`by the minimum number to the max number
<br>
<br>
The program assume the files to download are named such as:
<br>
`001.type` `002.type` `003.type` `004.type` `005.type`
<br>
It will do a loop until all files are downloaded in the download folder
